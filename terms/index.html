이용약관

이 페이지는 펠리카노 웹사이트와 관련 다운로드 안내에 적용되는 기본 이용 조건을 설명합니다.

1. 서비스 소개

펠리카노는 Windows용 로컬 우선 클립보드 매니저이며, 이 웹사이트는 프로그램 소개와 다운로드 링크 제공을 위한 정적 안내 페이지입니다.

2. 소프트웨어 제공 방식

펠리카노와 관련 웹페이지는 현재 제공되는 상태 그대로 제공됩니다. 기능, 배포 방식, 제공 여부는 예고 없이 변경되거나 중단될 수 있습니다.

3. 사용자 책임

클립보드에는 민감한 정보가 포함될 수 있습니다. 사용자는 자신의 기기 환경과 저장되는 데이터의 성격을 고려하여 프로그램을 사용해야 하며, 중요한 정보의 관리 책임은 사용자에게 있습니다.

4. 외부 링크

이 사이트는 GitHub 저장소, GitHub Issues, 설치 파일 다운로드 링크 등 외부 서비스로 연결될 수 있습니다. 외부 사이트와 서비스의 운영 및 정책은 각 제공자의 기준을 따릅니다.

5. 문의

서비스 관련 문의는 GitHub Issues를 통해 접수할 수 있습니다.
