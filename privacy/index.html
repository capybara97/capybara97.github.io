개인정보처리방침

이 페이지는 펠리카노 웹사이트와 프로그램 사용 과정에서 어떤 정보가 처리될 수 있는지에 대한 기본 원칙을 설명합니다.

1. 로컬 우선 원칙

펠리카노는 로컬 우선 방식의 Windows용 앱입니다. 기본적으로 클립보드 히스토리와 관련 설정은 사용자의 기기 안에 저장되며, 운영자가 이를 수집하기 위한 별도 서버 업로드를 기본 동작으로 두지 않습니다.

2. 웹사이트에서 처리될 수 있는 정보

이 사이트는 정적 웹페이지로 운영되며, 일반적인 웹 호스팅 과정에서 접속 로그나 다운로드 요청 정보가 호스팅 인프라 수준에서 처리될 수 있습니다. 이는 페이지 제공과 파일 전달을 위한 범위로 한정됩니다.

3. 프로그램의 네트워크 동작

프로그램은 버전 확인 또는 업데이트 안내를 위해 원격 매니페스트를 조회할 수 있습니다. 다만 현재 기준으로 회원가입, 사용자 계정, 광고 추적, 마케팅 분석 도구를 전제로 하지 않습니다.

4. 사용자의 제어

사용자가 프로그램에서 저장한 클립보드 데이터와 설정은 기본적으로 사용자 기기에서 관리됩니다. 중요한 정보가 포함될 가능성이 있는 경우, 사용자는 자신의 보안 정책에 맞게 사용 여부를 판단해야 합니다.

5. 문의

개인정보 처리와 관련한 문의는 GitHub Issues를 통해 접수할 수 있습니다.
